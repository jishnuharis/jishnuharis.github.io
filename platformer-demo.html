Quests of the Shadowland

A 2D side-scrolling action platformer with 10 hand-crafted levels, three enemy types (soldiers, enemies, zombies), bullets, grenades, loot boxes, coins, and a built-in level editor. Encrypted save system using Fernet. Built with Python + Pygame.

Enemy AI

Soldiers and zombies use a vision cone to detect the player. Zombies without ammo chase and bite (20 damage). Soldiers shoot bullets when in range.

Loot System

Collect med-kits, ammo-kits, and grenade-kits scattered across each level. Coins are worth 10 points each and stack toward your high score.

Save System

Progress is Fernet-encrypted and stored between sessions — current level, max level unlocked, and all-time high score.

Level Editor

A built-in tool for building and editing levels using the same tile set as the main game. Outputs pickled data files for the game engine.
