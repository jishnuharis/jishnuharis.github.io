Shadowrift: Endless Run

A from-scratch reconstruction of the procedural-world engine — terrain generated on the fly from layered noise, an infinite horizontal world, a full day/night colour cycle, turret & slime enemies, and a shield/dagger inventory — rebuilt in canvas + vanilla JS. The original repo is an active work-in-progress Pygame project; this demo takes its core systems (noise terrain, chunk-style streaming, day/night) and adds a basic health bar and combat loop on top, since the source project explicitly doesn't have those yet.

⚠ original repo is WIP — this demo extends it for playability

Shadowrift: Endless Run

Walk into an infinite, procedurally generated world. Survive turret fire and slime contact, collect a shield (blocks bullets) and a dagger (kills slimes), and explore as far as you like — the terrain never repeats and never ends.
